## 2.ความเข้าใจโจทย์ของฉัน

อธิบายโจทย์: ในโจทย์นี้จะให้เขียนโปรแกรมแสดงถึงว่าหมายเลขที่ห้องที่อยู่ปัจจุบันนี้ต้องพังกำแพงอย่างน้อยกี่ครั้งจึง
จะสามารถไปถึงห้องหมายเลข 1 ที่อยู่บนสุดได้ โดยปราสาทจะเป็นรูปแบบปิรามิด
โดยชั้นลงมาที่มากขึ้น จำนวนห้องที่อยู่ในแต่ละชั้นก็จะมากขึ้นตามลำดับ

Input: หมายเลขห้องที่อยู่ในปัจจุบัน 

Output: จำนวนครั้งที่น้อยที่สุดที่ต้องพังกำแพงเพื่อไปถึงห้องหมายเลข 1

Constraints: ในแต่ละชั้นที่ลงไปจำนวนห้องที่อยู่ในแต่ละชั้นก็จะมากขึ้นตามลำดับและบางชั้นอาจต้องพังกำแพงด้านข้างก่อนที่จะพังกำแพงเพื่อไปชั้นด้านบน

## 3.แผนแรกของฉัน

ขั้นแรก: สร้างฟังก์ชันสำหรับการหาค่าที่โจทย์ต้องการ โดย input หมายเลขห้องจะอยู่ในฟังก์ชันนี้ด้วย

ขั้นที่สอง: สร้างตัวแปร stage เบื้องต้นสำหรับการหาตำแหน่งชั้นของห้องที่เราอยู่

ขั้นที่สาม: สร้างตัวแปร value สำหรับเป็นขอบเขต (หมายเลขห้องสุดท้ายในชั้นนั้น) เบื้องต้น

ขั้นที่สี่: สร้าง while loop เพื่อหาว่าหมายเลขห้องปัจจุบันอยู่ที่ stage ไหน

ขั้นที่ห้า: สร้างเงื่อนไขของจำนวนครั้งการพังกำแพง

ขั้นที่หก: print output เป็นจำนวนครั้งในการพังกำแพงที่น้อยที่สุดเพื่อไปยังห้องหมายเลข 1
